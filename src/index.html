이 만들었습니다

Flutter (플러터)는 하나의 코드베이스로 모바일, 웹, 데스크톱, 임베디드 기기에서 네이티브로 컴파일되는 아름다운 애플리케이션을 만들기 위한 구글의 UI 툴킷입니다.

시작하기

다른 플랫폼 사용 경험이 있나요? 문서: iOS, Android, Web, React Native, Xamarin.

빠른 개발

Stateful Hot Reload를 이용하여 밀리초 만에 앱에 생기를 불어넣어보세요. 커스터마이징을 완전히 지원하는 다양한 위젯을 이용하여 몇 분만에 네이티브 인터페이스를 구축해보세요.

표현력 있고 유연한 UI

최종 사용자의 경험에 초점을 맞춰 신속하게 기능을 제공하세요. 계층형 아키텍처를 통해 완벽한 커스터마이징을 할 수 있기 때문에, 매우 빠른 렌더링과 표현력 있고 유연한 디자인을 구현이 가능합니다.

네이티브 수준의 성능

Flutter의 위젯은 스크롤링, 네비게이션, 아이콘과 폰트와 같은 모든 플랫폼의 중요 차이점을 통합합니다. 그리고 Flutter 코드는 다트의 네이티브 컴파일러를 사용하여 네이티브 ARM용으로 컴파일 됩니다.
